User defaults
  • In order for users to transact using the Stellar Extension their accounts need to have Testnet Lumens (TXLM) as a currency. Rehive's default setup is to add TXLM to your default group's primary account.
  • You can use the default setup provided on the right or choose your own.
  • Note: You must have multiple non-admin and non-extension groups to add them to the config.
