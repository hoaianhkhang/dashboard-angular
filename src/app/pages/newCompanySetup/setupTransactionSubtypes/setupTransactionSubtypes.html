Transaction subtypes
  • Subtypes are a way to mark specific transactions in order to categorize and sort them.
  • Select whether the subtype should be applied to a credit or debit transaction type.
  • Subtypes can be used to define business logic for marked transactions in the dashboard. E.g. A debit subtype named withdrawal can apply fees to only debit transactions marked with subtype withdrawal.
